# /optimization-plan

Turn GEO audit findings into an actionable plan. Split into quick wins (this week) and strategic investments (this quarter).

## Key optimization categories

### Content Structure
- Add FAQ schema to key pages
- Write definition-style content for "what is X" queries
- Create comparison pages vs competitors
- Publish original research/data that AI engines can cite

### Content Freshness
- Update stale pages (12+ months)
- Add recent data points and statistics
- Refresh URLs with current year

### Authority Building
- Get cited by authoritative sources
- Build structured data profiles
- Optimize Wikipedia and Wikidata entries

### Platform-Specific
- Chinese engines: publish on Baidu Zhidao, Zhihu, WeChat public accounts
- Western engines: optimize for AI-extractable structured content
