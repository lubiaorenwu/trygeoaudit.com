# Connectors

## How tool references work

Plugin files use `~~category` as a placeholder for whatever tool the user connects in that category. GEO Audit works with search and data tools via MCP.

## Connectors for this plugin

| Category | Placeholder | Included servers | Other options |
|----------|-------------|-----------------|---------------|
| AI search | `~~ai-search` | Tavily | SerpAPI, Brave Search |
| SEO tools | `~~SEO` | — | Ahrefs, SimilarWeb, Semrush, Moz |
| Analytics | `~~product analytics` | — | Google Analytics, Amplitude |
| Knowledge base | `~~knowledge base` | — | Notion, Confluence, Guru |
